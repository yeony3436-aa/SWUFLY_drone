# 📅 2026-04-06 실습: 기초 비행 및 LED 제어

- 드론의 위치 제어 모드와 LED 상태 제어를 학습
- 비행 중에 실시간으로 드론의 상태 시각화
<br>

## 🚀 주요 실습 내용

### 1. 사각형 비행 (`square_flight.py`)
- `ModeControlFlight.Position` 모드를 사용하여 정밀한 위치 제어를 수행
- 이륙 후 **0.5m 간격**으로 앞 -> 오른쪽 -> 뒤 -> 왼쪽 순서로 이동하여 사각형 궤적을 그리며 호버링 후 착륙
- `sendLostConnection` 설정을 통해 연결이 끊겼을 때의 안전 로직을 추가

<br>

### 2. LED 기본 제어 (`led.py`)
- 드론 본체 LED의 색상을 변경하고 효과를 주는 실습
- **BodyHold:** 지정한 색상(Cyan) 유지
- **BodyDimming:** 색상(Blue, Magenta)이 서서히 밝아졌다 어두워지는 효과 확인

<br>

### 3. LED 상태 비행 및 비상 착륙 (`led_flight.py`)
- 비행 상태에 따라 드론 LED 색상을 다르게 표시하여 현재 상태를 직관적으로 확인 가능
  - 🟢 **Green:** 포트 연결 완료
  - 🔵 **Blue:** 이륙 및 비행 시작
  - 🟣 **Purple:** 비행 경로 중간 지점 통과
  - 🟡 **Yellow:** 착륙 시작
  - 🔴 **Red:** 비상 상황 발생
- `keyboard` 라이브러리를 활용하여 비행 중 **Space 키** 입력 시 즉시 빨간색 LED를 점등하며 비상 착륙하는 안전 기능을 포함

<br>

## 🛠️ 사용 라이브러리
- `time`: 지연 시간 제어
- `keyboard`: 키보드 이벤트 감지 (비상 착륙용)
- `CodingDrone`: 바이로봇 드론 통신 프로토콜 제어
