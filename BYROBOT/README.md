# 🛸 BYROBOT Drone Coding Practice

바이로봇(BYROBOT)사의 **BRC-105** 드론을 활용하여 진행한 비행 제어 및 파이썬 코딩 실습 기록입니다. `CodingDrone` 라이브러리를 사용하여 드론의 움직임과 상태를 제어하는 과정을 정리합니다.

## 📋 실습 환경
- **Hardware:** BYROBOT BRC-105
- **Language:** Python
- **Library:** `CodingDrone`

## 📂 폴더 구성
- **[04_06](./04_06):** 기초 비행 및 LED 제어 실습
  - 사각형(Square) 궤적 비행
  - LED 색상 제어 (Hold, Dimming)
  - 비행 상태별 LED 피드백 및 비상 착륙 로직
